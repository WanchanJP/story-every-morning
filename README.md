# story-every-morning

「朝、目をひらくたびに。ひとつだけの物語が届く。」

このリポジトリは、AIが毎朝お届けするショートショート配信サービスの紹介ページ（ランディングページ）です。

## 概要

- Claude Code で HTML/CSS を生成
- ChatGPT との対話を通してプロジェクト構想・説明文などを整理
- 最初のデモとして GitHub Pages で公開中

## 使用技術

- 単一HTMLファイル
- レスポンシブ対応
- Mailフォームはプレースホルダー
